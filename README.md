#Conselheiro App

## Como usar

####Specs
- PHP 5.5.x
- MySQL 5.x
- jQuery 1.11 (ajax)

#### Steps
- Executar arquivo dump.sql localizado na pasta scripts;
- Executar arquivo seed.sql localizado na pasta scripts;

## Resetando a aplicação
- Faça truncate na tabela votes;
```
> TRUNCATE TABLE votes;
```
